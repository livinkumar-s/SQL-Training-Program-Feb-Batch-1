Joins 
---------
>> It is used to combine rows from two or more tables based on a related column.

Inner Join 
-------------
Returns only matching records from both table.
Ex: SELECT * FROM tb1 INNER JOIN tb2 ON (related columns)

Left Outer Join 
------------------
>> All the records from left table 
... matching records from right table will be returned.
Ex: SELECT * FROM tb1 LEFT JOIN tb2 ON (related columns)

Right Outer Join 
------------------
>> All the records from right table 
... matching records from left table will be returned.
Ex: SELECT * FROM tb1 RIGHT JOIN tb2 ON (related columns)

Full Outer Join 
-----------------
>> All records from both table 
right join   union     left join 

Cross Join
-----------
>> Returns every combination of rows.
no of rows in result --> rows in lt * rows in rt.
